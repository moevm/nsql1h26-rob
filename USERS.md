## Debug users (placeholders)

В приложении по умолчанию создаётся **1 пользователь** (если коллекция `users` пустая) из переменных окружения:

- `AUTH_DEFAULT_USER` (по умолчанию `admin`)
- `AUTH_DEFAULT_PASSWORD` (по умолчанию `admin`)
- `AUTH_DEFAULT_ROLE` (по умолчанию `admin`)

Если переменные не заданы, логин/пароль/роль будут `admin` / `admin` / `admin`.
